MyFlix

The server-side component (API) of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/:Title GET None A JSON object holding data about the selected movie containing description, genre, director and img. Example: 
              
                {
                  Title: '', 
                  Description: '',
                  Year: '', 
                  Genre: '',
                  Director: '',
                  Actors: '',
                  ImagePath: ''
                }
Return data about a genre by name/title /movies/genre/:Name GET None A JSON object holding data about the selected genre. Example: 
              
                {
                  name: '',
                  description: ''
                }
Return data about a director by name /movies/director/:Name GET None A JSON holding data about the selected director. Example: 
              
                {
                  name: '',
                  birth year: '',
                  death: '',
                  bio: ''
                }
Allow new users to register /users POST A JSON object holding data about the user to add. Example: 
              
                {
                  username: '',
                  password: '',
                  email: '',
                  birthday: '',
                  favorites: []
                }
A JSON object holding data about the added user, including an ID. Example: 
              
                {
                  id: '1'
                  username: '',
                  password: '',
                  email: '',
                  birthday: '',
                  favorites: []
                }
Allow users to update their user info /users/:Username PUT A JSON object holding the new user data. Example: 
              
                {
                  username: '',
                  password: '',
                  email: '',
                  birthday: '',
                  favorites: []
                }
A JSON object with updated user profile. Example: 
              
                {
                  id: '1'
                  username: '',
                  password: '',
                  email: '',
                  birthday: '',
                  favorites: []
                }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None A text message indicating the movie was added
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE None A text message indicating the movie was deleted
Allow existing users to deregister [users]/[username] DELETE :username None A message indicating the user was deregistered
